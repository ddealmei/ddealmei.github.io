## Course Description

This course introduces L3 computer-science students to the technical, social, and political stakes of cybersecurity. It gives enough technical background to reason about common threats while keeping the discussion connected to real systems, privacy issues, and everyday security decisions.

The practical sessions cover security monitoring, OSINT, passwords, password managers, and metadata leakage.

## Prerequisites

Basic computer-science knowledge.

## Teaching Language

French.
